# Version 2
- 小蜜蜂会动,从上往下慢慢飞
- 能控制小蜜蜂左右移动
- 入场算法
    - y轴要求负数,这样会形成慢慢的入场的效果, y = 0 - bee(height)
    - x轴要求有一定的富裕,即要去蜜蜂等移动物品不能贴边
        -基本下周的值为50起,最右边的计算方式应该等于,x(max)-bee(width)-50
    - 移动速度,x,y值
        - 主要考虑y值速度
        - 除飞机和蜜蜂外
        - 飞机的移动由上下左右键控制
        - 速度应该是一个TUPLE=(X,Y)
    - 方向问题
        - 具有移动方向由X,Y控制
        - 值只能是由-1,0,1三个控制
        - 应该是一个tuple()
        - 例如(-1,0)表示水平向左移动
        - (0,1)表示向下移动
